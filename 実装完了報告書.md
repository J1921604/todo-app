## 🚀 GitHub Pagesデプロイ準備

### ✅ 完了項目

1. **GitHub Actionsワークフロー**
   - ファイル: `.github/workflows/deploy.yml`
   - トリガー: `deploy`ブランチへのpush
   - プロセス: テスト → ビルド → デプロイ

2. **Vite設定**
   - ベースパス設定: `/todo-app/`
   - 本番環境用最適化済み

3. **package.json**
   - デプロイスクリプト: `npm run deploy`
   - gh-pagesパッケージ設定済み

### 🔧 手動セットアップが必要な項目

以下の手順は、GitHub上で**1回だけ**手動実行する必要があります：

#### 1. GitHub Pagesの有効化

**手順**:
1. GitHubリポジトリページを開く: https://github.com/J1921604/todo-app
2. 「Settings」タブをクリック
3. 左サイドバーから「Pages」をクリック
4. **Source**セクションで以下を設定:
   - **Source**: Deploy from a branch
   - **Branch**: `gh-pages` / `/(root)`
   - **Save**ボタンをクリック

#### 2. GitHub Actionsの権限設定（必要な場合）

**手順**:
1. リポジトリの「Settings」→「Actions」→「General」
2. 「Workflow permissions」セクションで:
   - **Read and write permissions**を選択
   - **Save**をクリック

#### 3. デプロイブランチの作成とプッシュ

**手順**:
```powershell
# デプロイブランチを作成
git checkout -b deploy

# GitHubにプッシュ（GitHub Actionsが自動起動）
git push origin deploy
```

#### 4. デプロイ完了確認

**手順**:
1. GitHubの「Actions」タブでワークフロー実行を確認
2. ワークフローが成功（✅グリーンチェック）になるまで待つ（2-3分）
3. 以下のURLでアプリにアクセス:
   ```
   https://j1921604.github.io/todo-app/
   ```
