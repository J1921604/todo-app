# 初級（React基礎レベルの人向け）

### 目的
- **コピー & ペーストで自分用のTodoページを作成する**    
- **サイドバーからページを開けるようにする**    

### 手順
1. **開発サーバーを起動**    
    - `npm run dev` で開発サーバーを起動
    - http://localhost:1234/ にアクセス
2. **サイドバーから新規ページ追加**    
    - サイドバーの「➕ 新規ページ追加」ボタンをクリック
    - 名前を入力して「追加」ボタンをクリック（アイコンは自動で📝に設定）
3. **開発サーバーを再起動**    
    - Ctrl+C でサーバーを停止
    - `npm run dev` で再起動
4. **動作確認**    
    - サイドバーに新しいページが表示されることを確認
    - ページを開いて、基本的な「追加／完了切り替え」が動くことを確認
※より詳しい情報については、下記マニュアルをご確認ください。  
`BASIC_TODO_GUIDE_JP.md`

### 主な機能
- ✏️ ページ名編集：サイドバーの✏️ボタンでページ名を変更可能
- 🗑️ ページ削除：サイドバーの🗑️ボタンでページを削除可能
  - **重要**: ページ削除時、そのページのタスクデータも完全に削除されます
  - 同じ名前で再度追加しても、以前のタスクは復元されません
- 📝 すべてのページで統一されたアイコン（📝）を使用

### 期待される成果
- サイドバーから自分の名前ページを開ける。    
- Todo追加・完了切り替えができる。    

### 成果物イメージ
サイドバーに、自分の名前ページが表示され、クリックすると、自分の名前ページを開ける。  
自分の名前ページは下記のように表示されている。  
※このページでできるもの：  
①タスクを入力して、追加ボタンを押下すると、タスクが追加できる。   
②追加されたタスクの前にある□をクリックすると、完了状態に変更できる。   
③完了済みタグをクリックして、完了したタスク一覧が表示される。

# 中級（React経験者向け）

### 目的
- **タスク削除機能を実装して、配列操作・状態管理を練習する**
### 実装内容
1. **個別削除機能**    
    - 各タスクに削除ボタンを追加。        
    - `filter` メソッドで対象タスクを削除。        
    - UIとlocalStorageが即時更新されることを確認。        
2. **一括削除機能**    
    - 「完了タスクをクリア」ボタンを追加。        
    - 完了済みタスクがある場合のみ表示。        
    - クリックすると完了タスクがすべて削除される。        
※より詳しい実装TIPSについては、下記マニュアルをご確認ください。  
`BASIC_TODO_GUIDE_JP.md`

**注意**: 既にテンプレートには削除機能が実装されています。このセクションは学習目的で記載されています。

### 学習ポイント
- Reactの状態管理（`useState`）    
- 配列操作（`filter`）    
- 条件レンダリング    
- localStorageとの同期   
### 期待される成果
- 各タスクを削除できる。    
- 完了タスクを一括削除できる。   
### 成果物イメージ
(作業する前の画面、既にいくつかのタスクが追加されている）  
 ①削除ボタンを押下する場合に、対象タスクが削除できる。  
 ②完了タスクがある場合、「完了タスクをクリア」ボタンが表示される  
 ③「完了タスクをクリア」ボタンを押下すると、完了したタスクがすべて削除できる。
